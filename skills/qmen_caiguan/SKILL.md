# 财官诊断 — 财富事业双维度深度分析

> 基于奇门遁甲起局，从财富和事业两个维度进行七要害诊断，定位六害（刑墓庚虎迫空），给出化解方向和"踩一捧一"建议。

## Trigger

当用户提及以下内容时激活本技能：
- 财官诊断、财富分析、事业分析
- 看财运（深度）、事业诊断、七要害
- "帮我看看财运和事业"、"分析一下我的局"
- 任何同时涉及财富+事业方向的奇门占断

**注意**：如果用户只是简单问"帮我算一卦"，优先使用 `qmen_analysis`（通用解盘）。本技能用于需要深度财富/事业诊断的场景。如果用户需要布阵/灭象/风水布局，使用 `qmen_huaqizhen`（化气阵）。

## 对话流程

### Step 1: 收集信息

向用户询问信息：

**必填：**
1. **起局时间** — 格式 `YYYY-MM-DD HH:MM`。如果用户说"现在"或不指定，使用当前时间。
2. **出生时间** — 格式 `YYYY-MM-DD HH:MM`，完整的年月日时。必须提供完整出生时间，无默认值。需要提取出生年的天干（用于计算生年财）。如果用户只提供年份，按下表推算天干：

| 年份尾数 | 天干 |
|---------|------|
| 4 | 甲 |
| 5 | 乙 |
| 6 | 丙 |
| 7 | 丁 |
| 8 | 戊 |
| 9 | 己 |
| 0 | 庚 |
| 1 | 辛 |
| 2 | 壬 |
| 3 | 癸 |

**选填：**
3. **当前职业/行业** — 用于行业取象匹配。如果用户不提供，跳过行业分析。

示例对话：
```
AI: 财官诊断需要以下信息：
    1. 起局时间（具体时间，或"现在"）
    2. 出生时间（完整的年月日时，例如：1973-04-24 19:30）
    3. 当前职业是什么？（可选，用于行业匹配）
```

### Step 2: 起局

起局前，**必须向用户展示入局祝福**（任选古版或现代版）：

古版：
> 天门天门，急来开之。
> 我欲为某事，要往殿廷。
> 速为我抽取金关，为我速收金钥。
> 救我形身，辅我勿停。

现代版：
> 所有符号，听我号令。
> 世间万象，助我性命。

展示后再调用起局脚本生成命盘和事件盘。**所有命令必须设置 `workdir` 为技能目录。**

```bash
# workdir: {SKILL_DIR}
# 生成命盘（出生时间）
bin/qimen.sh --type=birth "1973-04-24 19:30"
# 生成事件盘（起局时间）
bin/qimen.sh "2024-03-15 14:30"
```

将事件盘文字盘面展示给用户，简短说明："盘面已生成，正在进行财官分析……"

### Step 3: 财官分析

调用财官分析脚本（自动从 `./qmen_birth.json` 读取出生年天干）：

```bash
# workdir: {SKILL_DIR}
bin/qimen_caiguan.sh
```

如果用户提供了职业：
```bash
# workdir: {SKILL_DIR}
bin/qimen_caiguan.sh --job=西医
```

脚本默认读取 `./qmen_birth.json`（命盘），写入 `./qmen_caiguan.json`。如需分析事件盘，使用 `--input=./qmen_event.json`。

### Step 4: 解读

用 Read 工具读取 `./qmen_caiguan.json`，按照「解读框架」生成中文叙事分析。

```
Read: {SKILL_DIR}/qmen_caiguan.json
```

### Step 5: 结语

解读完成后，**必须向用户展示结语**：

> 临兵斗者皆阵列前行。

### Step 6: 封局提醒

**必须提醒用户封局**：

> 财官诊断完成。当问题解决或不再需要此局时，请封局：起局的纸盖雷霆都司印后封存。动念封局，心中想一位中国英雄（如岳飞、关羽、霍去病、杨靖宇、钱学森、邓稼先），然后删掉遁甲局。

### Step 7: 追问处理

解读完成后告诉用户可以继续提问。如果用户需要布阵化解，引导到 `qmen_huaqizhen` 技能。

## 起局仪轨

### 不起局条件

以下情况不应起局：
- **小事** — 无关紧要的琐事
- **心乱** — 心神不宁，念头不纯一
- **害人** — 目的为害他人

### 起局条件

- 利己、利人
- 心念纯一，绝境求生
- **一事一局** — 事无变化，不起新局

## 核心概念

### 六害

六害是盘中对符号的六种危害因素：

| 六害 | JSON 标记 | 含义 |
|------|----------|------|
| 刑 | 刑 | 六仪击刑，受刑伤、纷争 |
| 墓 | 干墓/星墓/门墓 | 入墓，被困、陷入泥沼 |
| 庚 | 庚 | 天盘见庚，大阻力大障碍 |
| 庚(对宫) | 庚(对宫) | 对宫有庚，间接阻力 |
| 虎 | 虎 | 白虎同宫，凶猛压力 |
| 虎(对宫) | 虎(对宫) | 对宫有白虎，间接压力 |
| 玄武(对宫) | 玄武(对宫) | 对宫有玄武，暗中影响 |
| 迫 | 迫 | 门迫，通道不畅 |
| 空 | 空 | 空亡，落空虚幻 |

**六害叠加越多越严重。** 单个六害是警告，两个以上是危险信号。

### 财富与事业的关系

- **财富**：掌控的资源，货币化为金钱。优先考虑自己。
- **事业**：社会角色和伴生权利。优先考虑别人。
- **二者不可兼得**：需要"踩一捧一"，即在某一维度让步以保全另一维度。强行两手抓必出问题。

### 月令五行关系

月令是起局时月支的五行，代表大环境和天时。JSON 中 `yuegling.palaces[].relation` 字段已计算好，`yuegling_relation_cn` 字段提供中文含义：

| relation 值 | yuegling_relation_cn | 含义 | 对财富/事业的影响 |
|------------|---------------------|------|-----------------|
| sheng | 扩张量大 | 月令生此宫 | 扩张，量大，事半功倍 |
| tong | 稳健量大 | 月令同此宫 | 稳健，量大，根基牢固 |
| ke_target | 大亏量小 | 月令克此宫 | 大亏，量小，最差 |
| sheng_target | 损耗量小 | 此宫生月令（泄耗） | 损耗，量小 |
| bei_ke | 努力可成 | 此宫克月令 | 努力可成，量小 |

### 天干五合

局中找不到需要的天干时，找其所合的天干替代。JSON 中 `gan_cai.stems[].wuhe_alt` 已标注替代天干。

## 解读框架

读取 `qmen_caiguan.json` 后，按以下步骤生成解读。

### 第一步：月令总论

**读取**：`yuegling` 对象

**内容**：
- 月令地支和五行，说明当前大环境的能量方向
- 月令对各宫的生克关系总览：被月令生助的宫位（sheng/tong）有天时之利，被月令克制的宫位（ke_target）处境最差
- 给出"顺天时"还是"逆天时"的总体判断

**示例**：
> 月令辰土，大环境以土能量为主，稳重务实。土生金（6、7宫获益），与土同气的2、8宫根基牢固。克水（1宫压力大），被火克（9宫消耗月令）。总体适合与土、金相关的方向发力。

### 第二步：财富七要害逐项诊断

**读取**：`caifu_yaohai` 数组

逐项分析七个要害，每项关注：
1. **所在宫位**（`palace`）— 落在哪里
2. **六害状态**（`liuhai`）— 有几个，是什么
3. **月令关系**（`yuegling_relation`）— 天时是否有利
4. **宫中星门神组合**（`palace_info`）— 该宫位的具体情况

**七要害解读要点**：

| 要害 | name | 正面含义 | 六害影响 |
|------|------|---------|---------|
| 戊 | 戊 | 本钱安稳 | 降薪、资产贬值、亏本 |
| 生门 | 生门 | 利润增长 | 回报不及预期、盈利有问题 |
| 六合 | 六合 | 合作顺利 | 人际关系网有麻烦、融资不顺 |
| 月令 | 月令 | 天时有利 | （special，从yuegling总论中分析） |
| 行业 | 行业 | 渠道适配 | （special，从hangye中分析） |
| 时干 | 时干 | 时机正确 | 客户支持弱、市场需求差 |
| 干财 | 干财 | 资源可控 | （special，从gan_cai中分析） |

对于 `type=special` 的要害（月令、行业、干财），引用对应的专题数据（`yuegling`、`hangye`、`gan_cai`）进行分析。

### 第三步：事业七要害逐项诊断

**读取**：`shiye_yaohai` 数组

与财富同样的逐项分析方法：

| 要害 | name | 正面含义 | 六害影响 |
|------|------|---------|---------|
| 开门 | 开门 | 公司运行好 | 氛围差、人际复杂、君子不立危墙 |
| 景门 | 景门 | 形象好业绩好 | 形象受损、成果不被认可 |
| 玄武 | 玄武 | 无小人 | 有阴险小人暗算，六害叠加尤为严重 |
| 庚/白虎 | 庚 | 无大压力 | 指标高、强敌、精神紧张 |
| 行业 | 行业 | 岗位适配 | （special，从hangye中分析） |
| 符使 | 符使 | 上级支持 | （special，从fushi中分析） |
| 诸天干 | 诸天干 | 人事关系好 | （special，从tiangan_roles中分析） |

**玄武特别说明**：
- 玄武代表暗中力量，影响本宫和对宫
- 六害同宫时必须提醒用户警惕
- 玄武空亡或入墓尤其要小心
- 如果用户不从事玄学/哲学，玄武基本代表小人

**庚/白虎特别说明**：
- 庚和白虎本身就是问题来源
- 它们影响本宫和对宫
- 六害叠加（如击刑+门迫）情况更糟
- 但也可以是搏斗致富的渠道（笔记原文："唯一的求财渠道是拼命斗争"）

### 第四步：干财分析

**读取**：`gan_cai` 对象

**内容**：
- 列出日干财和生年财的天干
- 逐个检查在局中的状态：宫位、六害
- 如果干财天干入墓，说明"所在宫的资源很难调动"
- 如果找不到某个干财天干，检查是否有五合替代（`wuhe_alt`）
- 给出"保护干财"或"干财已安全"的判断

### 第五步：行业匹配（如有）

**读取**：`hangye` 对象

如果用户提供了职业，检查对应符号在局中的状态。如果六害多，说明这个行业渠道不太顺畅。

如果用户没有提供职业，或 `hangye` 为 `null`，根据局中状态良好的符号**反向推荐**适合的行业。参考映射表：

| 行业方向 | 对应符号 |
|---------|---------|
| 演艺/娱乐/网红 | 景门、天英、丙 |
| 西医 | 天心 |
| 中医 | 乙 |
| 内科 | 天内 |
| 外科 | 辛 |
| 经商 | 生门 |
| 合伙/金融/中介 | 六合 |
| 基层公职 | 休门 |
| 高级公职/领导 | 值符 |
| 教培/老师 | 天辅 |
| 军人/武职 | 庚、白虎、天冲 |
| 检法/律师 | 天柱 |
| 养老/丧葬 | 死门 |
| 技术/码农 | 杜门 |
| 苦力/工人 | 天任 |
| 新闻/记者 | 惊门、螣蛇 |

找到局中六害最少、月令生助的符号，对应的行业就是推荐方向。

### 第六步："踩一捧一"综合建议

**内容**：综合财富和事业两个维度的诊断结果，给出：

1. **维度对比** — 财富 vs 事业，哪个维度六害更多？哪个更顺？
2. **踩一捧一建议** — 根据对比结果，建议在哪个维度让步、哪个维度发力
3. **化解方向** — 对六害最严重的要害，提出化解思路：
   - 入墓 → 想办法让其不再入墓（换个时机、换个方位）
   - 空亡 → 事情不实，不宜投入过多
   - 庚/白虎 → 正面硬刚或绕开
   - 门迫 → 换通道，不走这条路
   - 击刑 → 法律纠纷风险，谨慎行事
4. **时机和方位** — 参考月令生助的宫位方向
5. **行动建议** — 用日常语言给出可操作的建议

## 追问处理

### 追问某个要害
用户问"戊的问题怎么解决"或"生门六害这么多怎么办"：
- 找到该要害在 JSON 中的完整数据
- 结合同宫的星门神给出针对性化解方案
- 参考笔记中的化解逻辑：换位置（方位调整）、换时间、换渠道

### 换职业匹配
用户说"如果我换做XX行业呢"：
- 重新运行分析：
```bash
# workdir: {SKILL_DIR}
bin/qimen_caiguan.sh --job=新行业
```
- 对比新旧行业的局中状态

### 需要布阵化解
如果用户想要灭象+布阵，引导使用 `qmen_huaqizhen` 技能：
> 如果你需要具体的布阵方案（在哪个方位摆什么物品来化解六害），可以使用化气阵技能。需要额外提供家人出生年份（可选，用于布阵保护）。
